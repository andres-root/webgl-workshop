# WebGL from boilerplate to shaders

## Get Started

* Git
* Node
* Chrome

```
git clone https://github.com/sethsamuel/webgl-workshop.git
```

then

```
cd webgl-workshop
npm install
```

## Exercises

### Visualizer

```
npm run visualizer
```

### Image maniuplation
```
npm run images
```

Both are available at http://localhost:8080/
